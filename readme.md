# Landing Page - Force

Este projeto é uma **landing page** desenvolvida para apresentação de um sistema moderno e completo de gestão para **academias**. O objetivo principal desta página é atrair novos clientes (proprietários de academias e estúdios), detalhar as principais funcionalidades e vantagens do sistema, e facilitar o contato para uma demonstração ou contratação.

## 🚀 Sobre o Projeto

O sistema foi pensado para simplificar a vida dos donos de academia, e esta landing page atua como a vitrine principal do produto. Ela exibe desde os dores comuns que o sistema resolve, até os planos (pricing), dados estatísticos (metrics) e depoimentos.

O projeto é estruturado focado em:
- **Alta Conversão:** Design atraente voltado para gerar contatos (leads).
- **Interface Responsiva:** Adaptada para funcionar perfeitamente em PCs, tablets e smartphones.
- **Componentização:** Utilização de ferramentas atuais como Vite e React (dentro da pasta `force-app`) para facilitar iterações.
